This is a dull (by dull means no llm or RL support/module exists) quantitative, heuristic based risk + orchestration layer to fulfill rebalance and pass
other intents within a primary meta-aggregator vault based project for yeild optimization.

This pipeline is designed to be compatible with heartbeat/automation infrastructure like chainlink automation or gelato fully. Build is carried out in golang
with rest of the dependencies to make it faster executable and support concurrency in complex calls and EVM transactions to make it compatible for future 
architectural renovations and optimization/patching.

Primary role of this code base is to support native offchain infrastructure to the contract layer / protocol layer.
